BIENVENIDOS

Este proyecto se basa en un sistema de reconocimiento de voz lo que le permite al usuario ingresar una palabra a través de un micrófono que detecta poco ruido, la cual es comparada en un banco de registros. El sistema se controla por medio de GUI en Matlab, el cual se encarga de realizar el proceso, donde se reconoce el espectro de una señal de entrada y es procesada por la transformada rápida de Fourier la cual arroja el resultado en la comparación entre la señal de entrada con la que se encuentra en el banco de registro, el cual está en el sistema programado en Matlab. Esta gráﬁco de Matlab permite grabar, reproducir y comparar el espectro ingresado.
